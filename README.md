## Objective
Predict **Song Title** and **Artist** from a short lyrics snippet using NLP.

## Dataset
Spotify_Song_Dataset.csv  
Columns: artist, song, link, text

## Tech Stack
- Python
- scikit-learn
- NLTK

## How to Run
1. Place `Spotify_Song_Dataset.csv` inside `data/`
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run:
   ```bash
   python src/lyric_search.py
   ```
